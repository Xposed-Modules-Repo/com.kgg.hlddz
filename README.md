# 欢乐斗地主不看广告

欢乐斗地主看广告给奖励，需要看15秒视频后才能关闭领取，有了这个模块，再也不用忍受这种恶心的行为，
广告页面弹出1秒后自动关闭并领取奖励

本模块仅用于学习交流，请勿用于任何非法用途，否则后果自负！
